## Installation

Clone source code, install database script, build and run
    
## Run Locally

Clone the project

```bash
  git clone https://trongthang91@bitbucket.org/trongthang91/esm.git
```

Go to the project directory

```bash
  cd esm
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```


## Tech Stack

**Client:** React, CoreUI

**Server:** Java, Spring Boot

**Database:** MariaDB
